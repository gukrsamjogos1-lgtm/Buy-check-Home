 Buy Check Home
 

O "Buy check home"é um aplicativo voltado para cidades inteligentes que conecta clientes a vendedores de rua em tempo real. A proposta é permitir que usuários recebam notificações quando vendedores estiverem próximos.

 Objetivo

Facilitar a comunicação entre vendedores ambulantes e clientes, tornando a busca por produtos mais prática e rápida.

 Público-alvo
Clientes que desejam encontrar vendedores próximos
Vendedores de rua que querem aumentar suas vendas
 Funcionalidades
Cadastro de vendedores
Notificação de proximidade
Opção de seguir vendedores favoritos
Visualização de vendedores próximos
 Como funciona

O aplicativo utiliza a localização do usuário para identificar vendedores próximos. Quando um vendedor cadastrado estiver na região, o cliente recebe uma notificação.

 Tecnologias utilizadas

Este projeto foi desenvolvido apenas como conceito, sem uso de linguagem de programação.

 Cronograma

O desenvolvimento foi organizado utilizando ferramentas de planejamento como Trello.
